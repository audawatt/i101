About HTML


1. Tags are the building blocks of html. They always start and end with < and >, and usually come in pairs.
An element is everything contained from the start tag to the end tag.

2. In the html skeleton, the head tells the code how to appear, and does not show when the code is run.
Meanwhile, the body tells the html what elements should appear on the page.

3. Headers have several uses, such as keeping an organized page structure, making it more accessible to viewers,
and increasing search engine optimization.

4. We learned about a lot of new html elemtents:

  1. paragraph: <p> </p>
  2. header: <h> </h>
  3. head: <head> </head>
  4. body: <body> </body>
